# 🎲 DBD Randomizer Discord Bot

Create random **Survivor** and **Killer** builds for **Dead by Daylight** directly on Discord.

---

# ✨ Features

* 🎲 Survivor & Killer Build Randomizer
* 🔄 Reroll & Perk Lock System
* ⭐ Favorite Perks
* 👤 Favorite Characters
* 📖 Interactive Favorites Menu
* ⬅️ Multi-page Navigation
* ⭐ Toggle Favorite Perks
* 🏗️ Complete Build Management
  * ➕ Create Builds
  * 📚 Interactive Build Menu
  * ✏️ Rename Builds
  * 📝 Edit Perks
  * 📤 Share Builds
  * 🗑️ Delete Builds
  * 📥 Import Builds
* 🔍 Search Perks & Characters
* 📊 Global Statistics
* 🏆 Most Popular Perks
* 📚 Perk Database Statistics
* 🎁 Active DBD Reward Codes
* 🖼️ Reward Images for every Code
* 🔎 Smart Autocomplete
* 💾 MongoDB Storage
* ⚡ Interactive Buttons & Modals
* ✅ Updated for DBD 10.0.1

---

# 📜 Commands

## 🎲 Randomizer

* `/dbd roll`

---

## ⭐ Favorites

* `/dbd fav perks`
* `/dbd fav character`
* `/dbd fav clear`

---

## 🏗️ Builds

* `/dbd build menu`
* `/dbd build create`
* `/dbd build import`

---

## 📊 Statistics

* `/dbd stats`
* `/dbd perks`
* `/dbd search`

---

## 🎁 Utilities

* `/dbd codes`
* `/dbd info`
